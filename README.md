# 영수 게시판

## 구현 목표
> 1. 회원가입을 통한 유저 생성 및 로그인
> 2. 영수 사진첩 제작
> 3. 전체 사진 중,랜덤 확인 기능
> 4. 해당 사진 별 댓글 작성 / 수정 / 삭제 + 좋아요 기능
